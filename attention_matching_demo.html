Step 3: The Matching Process - Compute Q @ KT

We need to compute how similar each query is to each key using dot products.

üéØ Example: Query from Token 1 matching with all Keys

Query1: [0.8, 0.2, 0.1]

Match with Key1:

Query1 ¬∑ Key1 = [0.8, 0.2, 0.1] ¬∑ [0.6, 0.3, 0.2]

= 0.8√ó0.6 + 0.2√ó0.3 + 0.1√ó0.2 = 0.48 + 0.06 + 0.02 = 0.56

Match with Key2:

Query1 ¬∑ Key2 = [0.8, 0.2, 0.1] ¬∑ [0.4, 0.8, 0.3]

= 0.8√ó0.4 + 0.2√ó0.8 + 0.1√ó0.3 = 0.32 + 0.16 + 0.03 = 0.51
